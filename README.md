README.md# SIWES Management System

## Description
SIWES Management System is a web application developed to manage Students Industrial Work Experience Scheme (SIWES).

## Features
- Student Registration
- Supervisor Management
- Placement Management
- Logbook Management
- Evaluation Management
- Reports Dashboard
- Secure Login Authentication

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
